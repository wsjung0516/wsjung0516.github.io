1.     개요.
이 프로그램은 모 은행에서 프로젝트 시행중인 수출입물품제재(미국의 적대국가에 수출입을 못하도록 제한을 목적으로 함)기능으로, 수출입 문서가 OCR(Optical Character Recognition) 문서의 이미지 형태의 데이터에서 AI 기술 적용으로 이미지에 포함된 글자를 판독한 결과를 조회, 수정, 수/출입 제재항목 심사, 심사내용 결재 등의 핵심기능 수행하는 프로그램.
이 프로그램은 Angular Element 기술을 적용하여 실제 실행되는 프로그램이 한 개의 JavaScript ( xxx.js file)로 생성되어 필요한 프로젝트에서 바로 JavaScript Component처럼 적용할 수 있도록 개발함.
이 파일을 적용하는 프로젝트는 이 파일의 input/output, property, event 정보를 참조하여 바로 적용(Embed)할 수 있음.
개발기술은 Angular의 Elements Material, Cdk, 그리고 Reactive X Library인 RxJS를 활용하여 개발함.

2.     프로그램설명
-------- Dynamic Tabs
실제의 각 거래건수별로 국적과 물품이 틀리기 때문에 OCR 판독한 데이터가 약 20장 내외가 여러 세트(가로축 Tab)로 구분되고, 각 세트당 여러(세로축 Tab) 페이지의 OCR문서가 구분되어 접수됨
그러므로 프로그램에서는 각 거래건수별로 가로Tab, 세로Tab을 Dynamic하게 구성 해야함.
-------- 문서분류명,
분류된 각 OCR에 이미지에서 읽은 문서제목이 잘못 표기되어 세로 Tab 메뉴가 된 경우 이를 즉석에서 수정 반영하는 기능.
Dropdown기능, Editing기능, 메뉴 이동기능이 포함됨.
--------- Image Zoom in/Zoom out
각 세로Tab에 연결된 이미지는 Zoom-in/Zoom-out 기능 및 이미지에 특정 항목을 Rectangular로 Marking하는 기능이 포함됨.
Marking하는 이유는 문서에서 데이터가 추출된 위치를 알려주기 위함.
---------- Sanction Rule
OCR Image파일에서 추출한 제재항목을 표현하고, 잘 못 추출한 부분이 있으면 직접 입력하는 기능이 포함됨.
모든 페이지의 포함된 데이터, 각 페이지별로 포함된 데이터, 각 페이지의 특정 항목의 다른 페이지의 데이터와 중복된 경우 알람 기능 등이 포함됨.

1.     개요
이 프로그램은 본인이 처음부터 설계하고 디자인한 것이 아니라 Angular로 개발된 샘플프로그램 (Angular source: https://github.com/cypress-io/angular-pizza-creator)을 두명의 다른 개발자가 Redux(NgXS), RxJS를 조합하여 재 작성한 것인데, 이 프로그램이 (https://github.com/tommythongnguyen/Ngxs-Pizza-Order)미완성 상태로 정상 동작이 불가하여, 중요 부분을 수정한 것입니다. 원래는 DB와 연결되어 동작하지만, 온라인 데모용으로 Redux 기능으로 대신하여 동작합니다.
소스는 체계적으로 설계되어, 쉽게 내용을 파악할 수 있으며 유지보수가 쉽게 구성되어 있습니다.

2.     프로그램설명
New Pizza : 피자를 새로 생성할 수 있음.
View Pizza : 이미 만들어진 피자의 토핑을 바꾸거나 삭제할 수 있음.

3.   프로그램 소스 설명

pizza-item: 생성된 피자를 조회 하여 토핑 내용 수정 및 피자를 삭제 명령 기능.
pizza-toppings: 피자의 토핑을 선택하는 기능. 선택한 결과를 pizza-form에 전달함
